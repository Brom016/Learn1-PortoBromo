Cara Lanjutan
Matikan Lewat
Services & Registry
🛠️ Via Services
1.Tekan Win+R
2.Ketik services.msc
3.Cari Windows Update
4.Set Startup Type → Disabled
5.Klik Stop → Apply
🔑 Via Registry
1.Tekan Win+R
2.Ketik regedit
3.Masuk ke folder WindowsUpdate\AU
4.Buat DWORD NoAutoUpdate
5.Set value = 1, lalu Restart
⚠️ Registry untuk pengguna advance — hati-hati saat mengeditnya!
